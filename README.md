# **Enzyme Mechanics**

An interactive study resource developed alongside my coursework in mechanistic enzymology at UF. This repository integrates biochemical literature analysis with computational tools to explore enzyme mechanisms, experimental design, and kinetic concepts.

## Contents

Case Studies — mechanistic analysis of enzyme systems including the Klenow Fragment (DNA Pol I), Alcohol Dehydrogenase, and redox cofactor enzymes (Nicotinamides, Flavins, Cytochrome P-450s)
Experimental Methods — conceptual breakdowns of pre-steady state techniques including rapid quench-flow, pulse chase, stopped-flow, and radiolabeling
Molecular Visualization — 3D protein structure rendering via NGLview and 2D molecular visualization via RDKit

## _Tools Used:_
Python, Jupyter Notebook
RDKit, NGLview, pycairo
NumPy, Matplotlib

## _Status_
Actively in development alongside coursework. Kinetic simulations and expanded case studies are planned as the course progresses.
